
dfBlog WordPress theme version 1.1.4

Visit http://www.danielfajardo.com/dfblog/ for updates and extra packs.



Features
#####################################################################

	* Fixed width
	* Two columns
	* Right sidebar
	* 36 style combinations
	* Minimalist -- simple -- clean -- plain -- white -- red -- blue -- pink
	* Supports comments threading (WP 2.7), comments paging (WP 2.7), gravatars in comments, post tags and sidebar widgets.
	* Compatible with WordPress versions 2.5 to 2.7.1
	* The theme is not tested with older versions, still, there is no reason why it shouldn't work in older WP versions.
	* Valid XHTML and CSS

Installation
#####################################################################

	* Extract the dfBlog.zip file in a convenient location
	* Copy the 'dfBlog' folder to '/wp-content/themes/' directory in your web server.
	* Log in to the WordPress Administration Panels
	* Select the 'Themes' sub-menu from 'Appearance' menu (or 'Design'/'Presentation' depending on the WP version you are using).
	* Select the dfBlog theme from the available list of themes to preview
	* Activate the theme.
	* Your selection should now become active.

Configurations in the 'Theme options' sub-menu.
#####################################################################

	* You can:
		1. activate the logo as image or text
		2. change the menu label for the home page
		3. show or hide the 'Subscribe to Feed' of the Pages Menu
		4. select the skin for the header and/or the background
